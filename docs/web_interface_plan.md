# Web Interface Plan

## Purpose

This document defines the plan for adding a simple web interface to the AI Novel To Script project.

The current project can already run from the command line. Stage 4 will add a demo-friendly web interface so that users can convert novel text into scene-centered YAML through a browser.

---

## Design Goal

The web interface should be simple, stable, and easy to demonstrate.

It should allow users to:

1. Enter or paste novel text.
2. Select an extraction mode.
3. Convert the novel into YAML.
4. View the generated YAML result.
5. Download the YAML output.

---

```text
